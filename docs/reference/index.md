# Package index

## Classification of cases

Functions for designating cases as belonging to a specific type

- [`extreme_on_x()`](https://ingorohlfing.github.io/MMRcaseselection/reference/extreme_on_x.md)
  : Extremeness of cases on an independent variable
- [`extreme_on_y()`](https://ingorohlfing.github.io/MMRcaseselection/reference/extreme_on_y.md)
  : Extremeness of cases on the dependent variable
- [`pathway()`](https://ingorohlfing.github.io/MMRcaseselection/reference/pathway.md)
  : Pathway case
- [`predint()`](https://ingorohlfing.github.io/MMRcaseselection/reference/predint.md)
  : Classification of cases as typical and deviant using a prediction
  interval.
- [`residstd()`](https://ingorohlfing.github.io/MMRcaseselection/reference/residstd.md)
  : Classification of cases as typical and deviant using the standard
  deviation of the residuals.
